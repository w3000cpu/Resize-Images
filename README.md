# Resize-Images

Application to resize images using C#

This application is resizing the height, width, resolution of any images with the below formats:
  1. JPEG Files(JPEG, JPE, JPG).
  2. PNG Files(PNG, PNS).
  3. BMP Files(BMP, RLE, DIB).
  4. EMF Files(EMF).
  5. EXIF Files(EXIF).
  6. TIFF Files(TIFF, TIF).
  7. WMF Files(WMF).
  8. GIF Files(GIF).
  9. ICON Files(ICO).
